# 💡 Sunu-Idées

## Description du projet

Sunu-Idées est une boîte à idées numérique, anonyme et collaborative
développée sous forme de tableau de bord dynamique.
Elle permet à chaque membre d'une communauté (apprenants, formateurs, staff)
de proposer, enrichir et gérer des idées sans avoir besoin de se connecter.
Toutes les interactions se font sans rechargement de page grâce à la
manipulation dynamique du DOM.

---

## Technologies utilisées

- **HTML5** - Structure de la page
- **CSS3** - Mise en forme et styles personnalisés
- **JavaScript (Vanilla)** - Logique et manipulation du DOM
- **Bootstrap 5** - Composants UI et mise en page responsive
- **Bootstrap Icons** - Icônes
- **LocalStorage** - Persistance des données côté navigateur

---

## Fonctionnalités

- **Ajouter** une idée avec un titre, une catégorie et une description
- **Afficher** les idées sous forme de cartes dynamiques
- **Catégories** : Pédagogie, Événement, Vie de campus, Amélioration technique
- **Filtrer** les idées par catégorie
- **Modifier** le titre, la catégorie et la description d'une idée
- **Supprimer** une idée définitivement
- **Persistance** des données grâce au LocalStorage
