# 💡 Sunu-Idées

## Description du projet

Sunu-Idées est un système web collaboratif de type Single Page Application
permettant à chaque membre d'une communauté (apprenants, formateurs, staff)
de proposer, enrichir et gérer des idées de manière anonyme et instantanée.
Toutes les interactions se font sans rechargement de page grâce à la
manipulation dynamique du DOM et les données sont partagées en temps réel
grâce à Supabase.

---

## Technologies utilisées

- **HTML5** - Structure de la page
- **CSS3** - Mise en forme et styles personnalisés
- **JavaScript (Vanilla)** - Logique et manipulation du DOM
- **Bootstrap 5** - Composants UI et mise en page responsive
- **Bootstrap Icons** - Icônes
- **Supabase** - Base de données en ligne et temps réel
- **OpenRouter (Llama 3.3)** - Intelligence artificielle pour suggérer les catégories

---

## Fonctionnalités

- **Ajouter** une idée avec un titre, une catégorie et une description
- **Afficher** les idées sous forme de cartes dynamiques
- **Catégories** : Pédagogie, Événement, Vie de campus, Amélioration technique
- **Filtrer** les idées par catégorie
- **Rechercher** une idée par titre
- **Modifier** le titre, la catégorie et la description d'une idée
- **Supprimer** une idée avec confirmation
- **Suggérer** automatiquement une catégorie grâce à l'IA
- **Mode nuit/jour** pour le confort visuel
- **Persistance** des données grâce à Supabase
- **Collaboratif** : toutes les idées sont visibles depuis n'importe quel appareil
